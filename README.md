# todoWeb
  Servlet과 Jsp를 이용한 일정 관리 웹 어플리케이션입니다.<br/>
  JDBC를 통해 DB와 연동되어 있으며,<br/>
  **유저** - 가입, 프로필, 프로필 수정, 탈퇴 가능 <br/>
  **Todo(일정)** - 추가,보기,수정,삭제 가능 <br/>
  위와 같은 CRUD 기능을 구현하였습니다. <br/>
  <br/><br/>
  현재 스프링 프레임워크 적용 중이며 해당 프로젝트는 아래 주소에 저장하였습니다.<br/>
